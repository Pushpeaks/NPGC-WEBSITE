Library
Rules

Please adhere to the following rules and regulations to ensure a conducive learning environment for all library users.

Rules
01

Strict discipline and silence are to be observed in the Library.

02

Each user would be provided with library card for issue of books.

03

Issue of books for home – Each student would be issued six books at a time for a period of one semester. (JULY TO DEC & JAN TO JUNE).

04

Reissue of books by a user may be allowed maximum twice more after first issue for 10 days.

05

Failure to return books on Due-date would compulsorily lead to penalty of Re.10/- per month for all students.

06

A ‘No Dues Certificate’ for each student is to be obtained from the Librarian while withdrawing any student from college

07

Dictionaries, Encyclopaedia etc. would not be issued.

08

Newly purchased books will not be issued to any user for at least one month. But it will be displayed separately as new arrivals.

09

Issue of subject reference books- Only subject reference books from reference section would be issued for home use on those days when the successive day is a holiday. The book will have to be returned on the next working day.

10

In case a user loses a book he/she would have to either replace the book or deposit three times the price of the book in the library.

11

Magazines and periodicals may also be issued for a Week only.

12

Library books are to be deposited at least one week before leaving the station or any break (Short Vacation) and 15 days before any long vacation.

13

Each Student would be provided with one Library cards for issue of books. For this purpose, students should manage one Passport size recent photographs in School-Uniform.

14

All students should maintain a Book-Review-Note during Library period-study as per given format. For this purpose, a 40 pages Paperbound copy will be sufficient. Library Books can be issued and returned during Library Period of the class only and not in any teaching period.

Important Note

All library users are expected to follow these rules strictly. Any violation may result in suspension of library privileges. For any queries, please contact the Librarian.
